RIoT-Board
================

Boot Media
----------
Use the image ``platform-v7a/images/riotboard.hdimg`` and copy it onto your MicroSD-Card.

Serial Console
--------------
The serial boot console is available at the debug-header J18 at 115200 Baud and 8N1 with the following pinout:

* J18.1: TxD
* J18.2: RxD
* J18.3: GND
