Raspberry Pi 3
==============

Boot Media3
----------
Use the image ``platform-v7a/images/rpi3.hdimg`` and copy it onto your MicroSD-Card.

Serial Console
--------------
The serial boot console is available at J8 (the GPIO header) with 115200 Baud and 8N1 on the following Pins:

* J8.6: GND
* J8.8: Tx
* J8.10: Rx
